## 一、测试环境准备

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 安装 Python 3.8+ | `python --version` 显示正确版本 |
| 2 | 安装依赖：`pip install -r requirements.txt` | 所有依赖安装成功 |
| 3 | 编辑 `config.py`，修改 `PHOTO_ROOT` 和 `VIDEO_ROOT` 为实际路径 | 配置保存成功 |
| 4 | 启动服务：`python app.py` | 控制台显示启动信息，局域网 IP 地址 |
| 5 | 浏览器访问 `http://localhost:5000` | 页面正常加载 |
