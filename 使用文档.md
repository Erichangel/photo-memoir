## 项目简介

时光印记是一个基于Node.js的照片回忆录应用，让您可以通过时间线的形式回顾和展示珍贵的照片和视频回忆。

### 主要功能

- 📱 **用户选择页**：温馨的欢迎页面，支持多个用户
- 📅 **时间线浏览**：以时间线形式展示各个章节
- 🖼️ **媒体画廊**：精美的照片和视频浏览体验
- 💬 **弹幕系统**：为回忆添加互动评论
- 🎵 **背景音乐**：为回忆配上背景音乐
- 🎨 **精美UI**：暖粉色渐变、毛玻璃效果、流畅动画
- ⚙️ **管理后台**：完整的内容管理系统

### 基础环境

- **Node.js**: 14.0 或更高版本
- **npm**: 6.0 或更高版本
- **浏览器**: Chrome 90+ / Firefox 88+ / Safari 14+

### 可选环境

- **MongoDB**: 4.4 或更高版本（用于数据持久化）

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/Erichangel/photo-memoir.git
cd photo-memoir
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动项目

#### 开发模式（带自动重启）

```bash
npm run dev
```

#### 生产模式

```bash
npm start
```

### 4. 访问应用

- **前台页面**: http://localhost:3000
- **后台管理**: http://localhost:3000/admin.html

## 前台浏览端使用指南

### 用户选择页

1. **进入页面**：访问 http://localhost:3000
2. **选择用户**：点击用户卡片进入对应的回忆空间
3. **爱心动画**：背景会有爱心飘落动画，营造温馨氛围

### 时间线首页

1. **浏览时间线**：通过滚动查看所有章节
2. **章节节点**：每个节点显示日期范围、标题和统计信息
3. **继续回忆**：如果有浏览记录，会显示"上次看到这里"标记
4. **进入章节**：点击任意章节节点进入详情页
5. **音乐播放**：底部有音乐播放器，可以播放背景音乐
6. **导航返回**：点击左上角返回按钮（如存在）

### 章节详情页

#### 画廊操作

1. **查看大图**：左侧主区域显示当前选中的照片
2. **切换照片**：
   - 点击左侧缩略图列表
   - 点击大图左右箭头
   - 使用键盘左右方向键（如已实现）
3. **全屏查看**：点击大图进入全屏灯箱模式
4. **视频播放**：点击视频缩略图或大图播放视频

#### 文章区域

1. **阅读文章**：右侧显示回忆文章内容
2. **滚动浏览**：文章区域支持垂直滚动

#### 弹幕系统

1. **查看弹幕**：弹幕会从右侧飘过
2. **弹幕开关**：点击弹幕按钮可以显示/隐藏弹幕
3. **弹幕设置**：
   - 调节弹幕速度
   - 调节弹幕透明度
   - 调节弹幕显示区域
4. **发送弹幕**：在输入框输入内容，点击发送

#### 导航功能

1. **上/下一章**：点击底部导航按钮切换章节
2. **响应式布局**：在小屏幕上会自动调整为上下布局

## 后台管理端使用指南

### 仪表盘

1. **访问后台**：http://localhost:3000/admin.html
2. **查看统计**：首页显示照片、视频、章节、评论等统计数据
3. **快速操作**：
   - 点击"添加章节"创建新章节
   - 点击"上传照片"上传新照片
   - 点击"上传视频"上传新视频
4. **导航菜单**：左侧菜单可切换到各管理页面

### 章节管理

1. **查看章节列表**：以卡片形式显示所有章节
2. **编辑章节**：
   - 点击"编辑"按钮修改章节信息
   - 可修改标题、日期范围、启用状态
3. **启用/禁用**：切换开关控制章节是否在前台显示
4. **删除章节**：点击删除按钮移除章节（需确认）
5. **排序方式**：
   - 按开始时间排序
   - 按名称排序
6. **扫描文件夹**：扫描指定目录自动创建章节和导入文件

### 图片管理

1. **选择章节**：使用下拉菜单选择要管理的章节
2. **查看图片**：网格布局显示所有图片
3. **图片操作**：
   - **预览**：鼠标悬停显示操作按钮，点击预览图标
   - **重命名**：点击重命名图标，输入新文件名
   - **删除**：点击删除图标移除图片
4. **批量操作**：
   - 勾选多张图片
   - 显示批量操作栏
   - 支持批量删除和批量重命名

### 视频管理

1. **查看视频列表**：显示视频缩略图、时长和文件名
2. **视频操作**：
   - **播放**：点击播放图标弹出播放器
   - **重命名**：修改视频文件名
   - **删除**：移除视频文件
3. **批量操作**：支持多选后批量删除和重命名

### 文章编辑器

1. **选择章节**：选择要编辑的章节
2. **编辑工具栏**：
   - 加粗、斜体、标题等格式按钮
   - 插入Markdown语法
3. **实时预览**：左侧编辑，右侧实时渲染预览
4. **保存功能**：
   - 自动保存：编辑3秒后自动保存
   - 手动保存：点击保存按钮立即保存

### 音乐管理

1. **音乐列表**：显示所有背景音乐
2. **播放音乐**：点击播放按钮试听
3. **关联章节**：
   - 点击"设置"按钮
   - 选择要关联的章节
   - 支持全选（显示为"全部章节"）
4. **上传音乐**：点击上传按钮添加新音乐
5. **删除音乐**：移除不需要的音乐

### 评论管理

1. **评论列表**：显示所有评论，包含用户头像和内容
2. **章节筛选**：选择特定章节查看其评论
3. **删除评论**：点击删除按钮移除不当评论

### 用户管理

1. **用户列表**：显示所有用户卡片
2. **新建用户**：点击新建按钮添加用户
3. **编辑用户**：修改用户名和角色
4. **删除用户**：移除用户

### 系统设置

1. **基本设置**：配置项目基本信息
2. **互动效果**：
   - 爱心飘落开关
   - 打字机效果开关
   - 弹幕系统开关
3. **数据扫描**：扫描文件夹导入照片和视频

## API接口说明

### 用户接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/users | 获取用户列表 |
| POST | /api/users | 创建用户 |
| PUT | /api/users/:id | 更新用户信息 |

### 章节接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/chapters | 获取章节列表 |
| POST | /api/chapters | 创建章节 |
| GET | /api/chapters/:id | 获取章节详情 |
| PUT | /api/chapters/:id | 更新章节 |
| DELETE | /api/chapters/:id | 删除章节 |
| PUT | /api/chapters/:id/toggle | 切换启用状态 |

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/chapters/:id/photos | 获取章节照片 |
| POST | /api/photos/upload | 上传照片 |
| DELETE | /api/photos/:id | 删除照片 |
| PUT | /api/photos/:id/rename | 重命名照片 |
| POST | /api/photos/batch-rename | 批量重命名 |

### 视频接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/chapters/:id/videos | 获取章节视频 |
| POST | /api/videos/upload | 上传视频 |
| DELETE | /api/videos/:id | 删除视频 |

### 文章接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/articles/:chapter_id | 获取文章 |
| PUT | /api/articles/:chapter_id | 更新文章 |

### 音乐接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/music | 获取音乐列表 |
| POST | /api/music/upload | 上传音乐 |
| DELETE | /api/music/:id | 删除音乐 |
| PUT | /api/music/:id/chapters | 更新关联章节 |

### 评论接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/chapters/:id/comments | 获取章节评论 |
| POST | /api/chapters/:id/comments | 添加评论 |
| DELETE | /api/comments/:id | 删除评论 |

## 配置说明

### 环境变量配置

创建 `.env` 文件（参考 `.env.example`）：

```env
# 服务器配置
PORT=3000
NODE_ENV=development

# MongoDB配置（可选）
MONGODB_URI=mongodb://localhost:27017/photo-memoir

# 文件上传配置
UPLOAD_DIR=./uploads
MAX_FILE_SIZE=52428800

# 会话配置
SESSION_SECRET=your-secret-key-here
```

### MongoDB配置（可选）

1. 安装并启动MongoDB
2. 修改 `MONGODB_URI` 为您的MongoDB连接字符串
3. 重启服务器

**注意**：当前版本使用模拟数据，即使不配置MongoDB也可以正常运行。

## 常见问题

### Q: 如何修改前台页面的配色方案？

A: 编辑 `/workspace/public/index.html` 中的CSS样式，主要颜色变量在 `<style>` 标签中定义。

### Q: 上传的文件保存在哪里？

A: 默认保存在项目根目录的 `uploads` 文件夹中，可以通过 `UPLOAD_DIR` 环境变量修改。

### Q: 如何添加新用户？

A: 在后台管理端的"用户管理"页面点击"新建用户"按钮。

### Q: 可以在手机上使用吗？

A: 可以！项目已支持响应式设计，在手机和平板上都能正常使用。

### Q: 如何备份数据？

A: 如果使用MongoDB，可以使用 `mongodump` 命令备份。如果使用模拟数据，建议定期备份 `uploads` 文件夹。

### Q: 服务器启动失败怎么办？

A: 检查：
1. Node.js版本是否满足要求
2. 端口3000是否被占用
3. 依赖是否完整安装（运行 `npm install`）

### Q: 如何更新项目到最新版本？

A: 
```bash
git pull origin main
npm install
npm restart
```
